我们把强大的 Seedance 做成 Skill，在飞书聊天就能直接用：

  • 图生视频，文生视频，快试试你的 OpenClaw 加上 Seedance 的能力能玩出多少花！🤖


安装云 OpenClaw + 飞书聊天机器人教程








1. 前往火山引擎 一键部署OpenClaw

2. 点击购买部署



3. 套餐选择



4. 打开高级配置，设置Coding Plan API

如果没有开通方舟Coding Plan，可以先开通并创建API Key



5. 设置飞书 AppID和AppSecret

步骤1：在飞书开放平台创建机器人应用

  1. 登录飞书账号，进入开发者后台
  2. “创建企业自建应用”


  1. 选择 “按能力添加” -> 添加机器人



步骤2：进行飞书机器人的权限配置

a. 在左侧目录树中找到：开发配置 -> 权限管理

b. 选择“批量导入/导出权限”，复制右侧代码块中的JSON代码进行权限导入，给机器人开通im权限



这里权限包括：消息读取、飞书文档读取、创建和编辑飞书文档

c. 确认权限并申请开通

d. 后续可在“开通权限”中为机器人开通更多权限

步骤3：进行飞书机器人的事件和回调配置

  1. 在左侧目录树中找到“开发配置”->“事件与回调”


  1. 点击“订阅方式”旁的按钮添加事件配置


  1. 选择“使用 长连接 接收事件”


  1. 添加事件 -> 接收消息


步骤4：在机器人应用的凭证页面获取App ID和App Secret




步骤5：粘贴到创建实例的消息渠道集成当中

  1. 选择 飞书
  2. 粘贴对应的App ID和App Secret




步骤6：在飞书中使用OpenClaw

a. 确保应用已发布

b. 在飞书搜索框中搜索机器人应用的名称

c. 检查应用可用范围是否包含自己的账号




  1. 开通成功，后续也可以通过远程连接ECS Terminal快速修改模型和飞书APP的配置：
如果开通了coding plan，在 --ark-coding-plan 中填 "true"，未开通填 "false"
curl -fsSL https://openclaw.tos-cn-beijing.volces.com/setup.sh | bash -s -- \
    --ark-coding-plan "true" \
    --ark-api-key "替换为自己的方舟API Key" \
    --ark-model-id "doubao-seed-code" \
    --feishu-app-id "替换为飞书APP ID" \
    --feishu-app-secret "替换为飞书APP Secret"
更多教程参考 https://www.volcengine.com/docs/6396/2189942







安装 Seedance 生视频 skill








前置条件：安装 Clawhub skill，后面通过这个来按照其他skills。

  1. 检查当前是否已经安装，如果没有安装可以直接点击安装；

方式1：通过 OpenClaw Web控制台点击安装，安装完成后页面如下：




方式2：通过远程连接ECS Terminal安装：



npm install -g clawhub




  1. 安装 seedance-video skill

方法1：推荐

  2. 放在本地OpenClaw skill目录：/Users/your_user_name/.openclaw/skills/

安装完成后在OpenClaw Web控制台: http://127.0.0.1:18789/skills 可以查看这个skill：




  1. 在火山方舟 系统管理 （https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey?apikey=%7B%7D）中获取API Key



点击复制API key




  1. 通过远程连接登录ECS Terminal，通过命令行配置 API Key：















export ARK_API_KEY='your-api-key-here'

在ECS命令行Terminal中设置API Key：
