OpenClaw 公开文档里，`agents.defaults.compaction` 下最关键的是这几个：

- `enabled`

- `reserveTokens`

- `keepRecentTokens`

- `reserveTokensFloor`

文档给出的 Pi 压缩设置示例是：

```json
{
  "agents": {
    "defaults": {
      "compaction": {
        "enabled": true,
        "reserveTokens": 100000,
        "keepRecentTokens": 20000
      }
    }
  }
}
```

### 这些字段分别是什么意思

- `reserveTokens`：  
  给当前上下文窗口预留多少 token 头部空间。  
  实际上，OpenClaw 会在“上下文已使用 token 接近 `contextWindow - reserveTokens`”时进入压缩区间。文档明确说，OpenClaw 在会话接近或超过模型上下文窗口时触发自动压缩。

- `keepRecentTokens`：  
  压缩时保留最近多少 token 的原始消息，不把这部分也一起摘要掉。这个不是“触发阈值”，而是“压缩后保留多少最近上下文”。
